Петров Антон 181-351. Шифровка со сдвигом 2. Только на английском языке.
Как пользоваться шифратором:
1.Выберите что хотите сделать:
 1.Зашифровать. 
 2.Дешифровать.
2.Введите соответсвующую цифру. Нажмите Enter.
3.Введите текст который необходимо зашифровать/дешифровать, длиной не более 257 символов(с учетом пробелов и знаков препинания).
Нажмите Enter. На экран выведется готовый текст.
Чтобы выйти из программы нажмите 3 и Enter.
